# git-recent

List recently checked out branches. Parses the reflog to show branches you've worked on, ordered by most recent.

## Options

| Option | Description |
|--------|-------------|
| `--help`, `-h` | Show help message |
| `--version`, `-v` | Show version information |

## Examples

```bash
# List recently checked out branches
git recent
```
