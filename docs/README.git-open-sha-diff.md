# git-open-sha-diff

Open the diff view for a commit on GitHub in your browser.

## Options

| Option | Description |
|--------|-------------|
| `--help`, `-h` | Show help message |
| `--version`, `-v` | Show version information |

## Examples

```bash
# Open current commit diff
git open-sha-diff
```
