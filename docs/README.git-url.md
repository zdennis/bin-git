# git-url

Get the GitHub URL for the repository.

## Options

| Option | Description |
|--------|-------------|
| `--help`, `-h` | Show help message |
| `--version`, `-v` | Show version information |

## Examples

```bash
git url
# Output: https://github.com/zdennis/bin-git
```
