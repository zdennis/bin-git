# git-open-sha

Open a commit on GitHub in your browser.

## Options

| Option | Description |
|--------|-------------|
| `--help`, `-h` | Show help message |
| `--version`, `-v` | Show version information |

## Arguments

| Argument | Description |
|----------|-------------|
| `sha` | Commit SHA (default: current HEAD) |

## Examples

```bash
# Open current commit
git open-sha

# Open specific commit
git open-sha abc1234
```
