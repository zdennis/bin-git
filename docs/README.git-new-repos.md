# git-new-repos

Open GitHub's "create new repository" page in your default browser.

## Options

| Option | Description |
|--------|-------------|
| `--help`, `-h` | Show help message |
| `--version`, `-v` | Show version information |

## Examples

```bash
# Open new repo page
git new-repos
```
