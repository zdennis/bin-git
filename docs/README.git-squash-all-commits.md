# git-squash-all-commits

Squash all commits on the current branch into one. Useful for cleaning up history before merging.

## Options

| Option | Description |
|--------|-------------|
| `--help`, `-h` | Show help message |
| `--version`, `-v` | Show version information |

## Examples

```bash
git squash-all-commits
```
