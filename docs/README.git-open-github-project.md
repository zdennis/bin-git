# git-open-github-project

Open a GitHub project in your browser using shorthand notation. Takes `org/repo` format.

## Options

| Option | Description |
|--------|-------------|
| `--help`, `-h` | Show help message |
| `--version`, `-v` | Show version information |

## Arguments

| Argument | Description |
|----------|-------------|
| `org/repo` | GitHub organization and repository name |

## Examples

```bash
git open-github-project rails/rails
git open-github-project ruby/ruby
```
