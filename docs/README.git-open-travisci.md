# git-open-travisci

Open Travis CI builds for the repository in your browser.

## Options

| Option | Description |
|--------|-------------|
| `--help`, `-h` | Show help message |
| `--version`, `-v` | Show version information |

## Examples

```bash
git open-travisci
```
