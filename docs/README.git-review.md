# git-review

Review commits not yet pushed to remote. Shows the diff of local commits that haven't been pushed upstream.

## Options

| Option | Description |
|--------|-------------|
| `--help`, `-h` | Show help message |
| `--version`, `-v` | Show version information |

## Examples

```bash
git review
```

- [git-rreview](README.git-rreview.md) - Review remote commits not in local
