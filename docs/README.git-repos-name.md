# git-repos-name

Output the repository name in `org/repo` format from the remote URL.

## Options

| Option | Description |
|--------|-------------|
| `--help`, `-h` | Show help message |
| `--version`, `-v` | Show version information |

## Examples

```bash
git repos-name
# Output: zdennis/bin-git
```
