# git-whatadded

Find the commit that added a specific file.

## Options

| Option | Description |
|--------|-------------|
| `--help`, `-h` | Show help message |
| `--version`, `-v` | Show version information |

## Arguments

| Argument | Description |
|----------|-------------|
| `file` | File path to search for (required) |

## Examples

```bash
git whatadded path/to/file.rb
```
