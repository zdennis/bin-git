# git-open-circleci

Open CircleCI builds for the current branch in your browser.

## Options

| Option | Description |
|--------|-------------|
| `--help`, `-h` | Show help message |
| `--version`, `-v` | Show version information |

## Examples

```bash
# Open CircleCI for current branch
git open-circleci
```
