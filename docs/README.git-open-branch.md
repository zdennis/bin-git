# git-open-branch

Open the current branch on GitHub in your browser.

## Options

| Option | Description |
|--------|-------------|
| `--help`, `-h` | Show help message |
| `--version`, `-v` | Show version information |

## Arguments

| Argument | Description |
|----------|-------------|
| `branch` | Branch name (default: current branch) |

## Examples

```bash
# Open current branch
git open-branch

# Open specific branch
git open-branch feature/new-feature
```
