# git-rib

Interactive rebase on main branch. A shortcut for `git rebase -i main`.

## Options

| Option | Description |
|--------|-------------|
| `--help`, `-h` | Show help message |
| `--version`, `-v` | Show version information |

## Examples

```bash
git rib
```
