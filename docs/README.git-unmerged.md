# git-unmerged

Show branches with commits not merged to main. Useful for finding work that hasn't been integrated yet.

## Options

| Option | Description |
|--------|-------------|
| `--help`, `-h` | Show help message |
| `--version`, `-v` | Show version information |

## Examples

```bash
git unmerged
```
