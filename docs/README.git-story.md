# git-story

Output story/ticket ID from branch name. Extracts ticket numbers (like JIRA IDs) from branch naming conventions.

## Options

| Option | Description |
|--------|-------------|
| `--help`, `-h` | Show help message |
| `--version`, `-v` | Show version information |

## Examples

```bash
# On branch "feature/APPS-1234-add-login"
git story
# Output: APPS-1234
```
