# git-open-tree

Open the repository tree on GitHub in your browser.

## Options

| Option | Description |
|--------|-------------|
| `--help`, `-h` | Show help message |
| `--version`, `-v` | Show version information |

## Arguments

| Argument | Description |
|----------|-------------|
| `ref` | Branch, tag, or commit (default: current branch) |

## Examples

```bash
# Open current branch tree
git open-tree

# Open specific ref
git open-tree main
```
