# git-open-pulls

Open the GitHub pull requests page for the current repository in your browser.

## Options

| Option | Description |
|--------|-------------|
| `--help`, `-h` | Show help message |
| `--version`, `-v` | Show version information |

## Examples

```bash
git open-pulls
```
