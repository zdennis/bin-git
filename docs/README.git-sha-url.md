# git-sha-url

Get the GitHub URL for a specific commit SHA.

## Options

| Option | Description |
|--------|-------------|
| `--help`, `-h` | Show help message |
| `--version`, `-v` | Show version information |

## Arguments

| Argument | Description |
|----------|-------------|
| `sha` | Commit SHA (default: HEAD) |

## Examples

```bash
# URL for current commit
git sha-url

# URL for specific commit
git sha-url abc1234
```
