# git-rubocop-branch

Run rubocop on files changed in current branch vs main. Useful for checking only the files you've modified.

## Options

| Option | Description |
|--------|-------------|
| `--help`, `-h` | Show help message |
| `--version`, `-v` | Show version information |

## Examples

```bash
git rubocop-branch
```
