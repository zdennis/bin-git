# git-project

Initialize a new git project. Creates a directory, initializes a git repository, creates a README, and makes an initial commit.

## Options

| Option | Description |
|--------|-------------|
| `--help`, `-h` | Show help message |
| `--version`, `-v` | Show version information |

## Arguments

| Argument | Description |
|----------|-------------|
| `project-name` | Name of the project/directory to create (required) |

## Examples

```bash
# Create a new project
git project my-new-project
cd my-new-project
```
