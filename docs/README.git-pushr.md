# git-pushr

Push the current branch and set upstream to origin. Useful for first-time push of a new branch.

## Options

| Option | Description |
|--------|-------------|
| `--help`, `-h` | Show help message |
| `--version`, `-v` | Show version information |

## Examples

```bash
# Push current branch and set upstream
git pushr
```

Equivalent to: `git push --set-upstream origin <current-branch>`
