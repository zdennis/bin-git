# git-remote-branch

Output the remote tracking branch for the current branch (e.g., `origin/feature/x`).

## Options

| Option | Description |
|--------|-------------|
| `--help`, `-h` | Show help message |
| `--version`, `-v` | Show version information |

## Examples

```bash
git remote-branch
```
