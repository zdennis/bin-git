# git-rbranch

Alias for [git-remote-branch](README.git-remote-branch.md). Output the remote tracking branch for the current branch.

## Options

| Option | Description |
|--------|-------------|
| `--help`, `-h` | Show help message |
| `--version`, `-v` | Show version information |

## Examples

```bash
git rbranch
```
