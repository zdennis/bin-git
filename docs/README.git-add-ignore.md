# git-add-ignore

Download and append gitignore templates from GitHub's [gitignore repository](https://github.com/github/gitignore) to your `.gitignore` file.

## Options

| Option | Description |
|--------|-------------|
| `--help`, `-h` | Show help message |
| `--version`, `-v` | Show version information |

## Arguments

| Argument | Description |
|----------|-------------|
| `language` | The gitignore template name (e.g., Ruby, Python, Node) |

## Examples

```bash
# Add Ruby gitignore
git add-ignore Ruby

# Add Python gitignore
git add-ignore Python

# Add Node gitignore
git add-ignore Node
```
