# git-bin-git-path

Print the path to the bin-git directory where these git tools are installed. Useful for debugging PATH issues or finding script locations.

## Options

| Option | Description |
|--------|-------------|
| `--help`, `-h` | Show help message |
| `--version`, `-v` | Show version information |

## Examples

```bash
# Get the bin-git path
git bin-git-path
# Output: /path/to/bin-git/bin
```
