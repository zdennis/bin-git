# git-tree-url

Get the GitHub tree URL for the current branch or ref.

## Options

| Option | Description |
|--------|-------------|
| `--help`, `-h` | Show help message |
| `--version`, `-v` | Show version information |

## Arguments

| Argument | Description |
|----------|-------------|
| `ref` | Branch, tag, or commit (default: current branch) |

## Examples

```bash
# URL for current branch tree
git tree-url

# URL for specific ref
git tree-url main
```
